Research

Applying Artificial Intelligence and Machine Learning to solve real-world problems in education and society.

Research Projects

Adapting LLM for Regional Language Processing

Objective: Adapt LLMs using fine-tuning for culturally-aware Marathi NLP tasks.

Outcome: Built Marathi_Para, Marathi_Class, Marathi_Senti with up to 90% accuracy and improved fluency.

Adapting LLM for Question Paper Generation

Objective: Use RAG with syllabus data to generate Bloom’s Taxonomy-aligned papers.

Outcome: Produced structured, syllabus-aligned exam papers outperforming generic models.

Adapting LLM for Student Assessment Evaluation

Objective: Use embeddings and cosine similarity for semantic answer grading.

Outcome: High agreement with human grading and low RMSE.

Adapting LLM for Student Support System

Objective: Integrate RAG with university data for multilingual support.

Outcome: Reduced hallucinations and improved accuracy using verified documents.
